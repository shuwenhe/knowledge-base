# jinlei
# 项目描述：金雷科技大模型维修知识问答系统
# 项目目标
# 构建一套面向金雷科技内部使用的 智能维修知识问答系统，通过本地大模型结合企业私有知识库，实现对维修问题的精准回答与建议输出。
# 系统功能设计
# 0. 本地大模型推理（Qwen3 + Ollama）
# 使用 Ollama 部署并调用本地 Qwen3 大模型，确保数据不外泄、响应快速、可离线运行。
# 1. 文档读取与知识抽取（LangChain）
基于 LangChain 框架读取企业内部的 PDF / Word 维修文档。
自动解析设备说明书、维修流程、常见故障分析、操作规程等内容。
# 2. RAG 检索增强生成（Retrieval-Augmented Generation）
采用 RAG 架构将大模型与企业知识库结合。
查询时先精准检索相关知识，再由大模型生成最终回答，确保答案专业、可溯源。
# 3. 向量知识库（FAISS）
使用 FAISS 构建本地向量数据库，存储文档 Embedding。
支持快速相似度检索，保证大规模维修文档的高效查询。
# 4. Web 端交互界面
开发一个简洁、易用的 Web 查询页面：
输入维修问题
系统返回解析内容与推荐处理方案
支持查看引用文档片段，提高可解释性
# 5. 智能维修问答与建议生成
根据用户输入的维修问题，系统自动：
检索相关资料
提取关键信息
由大模型生成 专业维修建议 / 故障定位 / 操作步骤
